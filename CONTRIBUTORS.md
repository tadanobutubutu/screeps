## 🛠️ 貪献方法

### 人間の貪献者も歓迎！

1. Fork this repository
2. Create your feature branch
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

### Botの追加

新しいBotを追加したい場合:

1. `.github/workflows/`に新しいワークフローを作成
2. ユニークな`user.name`と`user.email`を設定
3. 適切な頻度でスケジュール
4. このファイルを更新
