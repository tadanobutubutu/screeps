# 😊 Creep Emotion System

あなたのCreepたちは感情を持っています！

## ✨ 機能

### 1. 🎭 豊かな表情システム

Creepたちは30種類以上の絵文字で感情を表現：

#### 😊 ポジティブ感情
- 😊 幸せ
- 🤩 興奮
- 😎 誇らしげ
- 😍 愛
- ⚡ エネルギッシュ
- ⭐ 成功

#### 😐 中立感情
- 🛠️ 作業中
- 🤔 考え中
- 🎯 集中
- 🚶 移動中

#### 😔 ネガティブ感情
- 😫 疲れた
- 😵 混乱
- 😟 心配
- 🤕 ダメージ
- 😨 動けない
- 🍔 お腹が空いた

#### 💪 活動ベース
- ⛏️ 採掘中
- 🛠️ 建設中
- ⬆️ アップグレード中
- 🔧 修理中
- 💊 治療中
- ⚔️ 戦闘中
- 🚚 運搬中

#### 🎉 特別
- 🎉 誕生日
- 🎆 レベルアップ
- 🎊 お祝い

### 2. 🎭 気分システム

Creepたちは1～5レベルの気分を持ちます：

- **5: とても幸せ** 😄
- **4: 幸せ** 😊
- **3: 普通** 😐
- **2: 悲しい** 😟
- **1: とても悲しい** 😭

#### 気分が変わる要因

✅ **ポジティブ**
- エネルギー満タン (+1)
- 他のCreepと交流 (+0.5)
- 達成感 (+3)
- 1500ティック生きる (+3)

❌ **ネガティブ**
- エネルギー不足 (-1)
- ダメージ受けた (-2)
- 動けない (-1)

### 3. 🎭 性格システム

各Creepは独自の性格を持ちます：

- **cheerful** (明るい) - よく笑う
- **serious** (真面目) - 仕事第一
- **energetic** (元気) - アクティブ
- **calm** (穏やか) - 落ち着いている
- **curious** (好奇心旺盛) - 探索好き
- **determined** (意志が強い) - 諸めない

### 4. 🏆 達成システム

Creepたちは達成を記録：

- 1500ティック生存 🎉
- 特別なイベント ⭐
- レベルアップ 🎆

### 5. 👥 ソーシャル機能

Creepどうしは交流します：

- 近くのCreepと挨拶 👋
- 一緒にいると幸せに 😊
- 50ティックごとに交流チャンス

### 6. 💪 パフォーマンスボーナス

幸せなCreepはよく働きます！

- **とても幸せ (5)**: +10% ボーナス
- **幸せ (4)**: +5% ボーナス
- **普通 (3)**: 通常
- **悲しい (2)**: -5% ペナルティ
- **とても悲しい (1)**: -10% ペナルティ

## 🎮 使い方

### 自動実行

自動で動きます！

- 全Creepが吸き出しに絵文字を表示
- 状況に応じて絵文字が変わる
- 100ティックごとに感情統計を表示

### コンソールコマンド

#### 個別のCreepを確認

```javascript
checkEmotion('harvester_12345')
```

**表示内容**：
```
🤖 Creep Emotion Report
Name: harvester_12345
Mood: Happy 😊
Current Emotion: ⛏️
Personality: cheerful
Age: 523 ticks
Achievements: 0
```

#### 全体の感情統計

```javascript
emotionStats()
```

**表示内容**：
```
😊 Emotion Statistics:
Very Happy: 3 😄
Happy: 5 😊
Neutral: 2 😐
Sad: 0 😟
Very Sad: 0 😭
Total Creeps: 10
```

## 📊 統計情報

メインループで自動記録：

```javascript
Memory.creeps[name].emotions = {
  mood: 4,                    // 1-5
  lastEmotion: '😊',        // 最後の絵文字
  experiencePoints: 0,        // 経験値
  achievements: [],           // 達成リスト
  personalityTraits: 'cheerful', // 性格
  birthTick: 12345           // 生まれたティック
}
```

## 📚 APIリファレンス

### EmotionSystem.display(creep)

Creepの感情を更新して表示。

### EmotionSystem.updateEmotion(creep)

状況に応じて感情を更新。

### EmotionSystem.celebrate(creep, achievement)

達成を祝う。

### EmotionSystem.interact(creep1, creep2)

2体のCreepを交流させる。

### EmotionSystem.getStats()

全Creepの感情統計を取得。

### EmotionSystem.checkCreep(creepName)

特定Creepの詳細情報を表示。
