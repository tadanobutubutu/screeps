## 🤖 機能

### 🚀 完全自動化
- **23の自動化ボット**: コードを24時間365日メンテナンス
- **AI搭載**: GitHub Copilot統合によるインテリジェントな修正
- **メンテナンス不要**: すべて自動で稼働
- **永久無料**: コストなし、制限なし

### 📈 リアルタイム監視
- **ゲームステータス**: 15分ごとのライブ統計
- **コンソールログ**: GitHubで完全アクセス
- **エラー検出**: 30分以内に自動検出
- **パフォーマンス**: 継続的な最適化

### 🔒 セキュリティ
- **5層防御**: 複数のセキュリティスキャン
- **CodeQL解析**: 高度な脅威検出
- **依存関係スキャン**: 脆弱性チェック
- **シークレット検出**: 漏洩防止

## 🚀 クイックスタート

1. **GitHub接続**（必須）
   - Screeps.com → 設定 → Git
   - `tadanobutubutu/screeps` に接続
   - ブランチ: `main`

2. **APIトークン**（任意）
   - 統計とログ用
   - GitHub Secretsに `SCREEPS_PROD_TOKEN` を設定

3. **スポーン＆リラックス**
   - あとはすべて自動です！

## 📊 プロジェクト統計

- **ロール**: 8つのクリープロールを実装
- **自動化**: 100%
- **稼働時間**: 24時間365日
