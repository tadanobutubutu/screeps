# 🎮 Gamification Guide

**開発をゲームに変えよう！**

## 🎯 概要

このリポジトリには、開発をより楽しくするためのゲーミフィケーションシステムが組み込まれています！

- 📈 **レベルシステム**: 貢献でポイント獲得、レベルアップ
- 🏆 **実績システム**: マイルストーンで実績アンロック
- 🎯 **デイリークエスト**: 毎日のチャレンジ
- 📊 **統計ダッシュボード**: 進捗可視化

## 📈 レベルシステム

### ポイント獲得方法

| アクション | ポイント | 絵文字 |
|----------|----------|--------|
| Issueを解決 | **10** | 🎯 |
| PRをマージ | **20** | 🚀 |
| コミット | **1** | 💻 |
| デイリークエスト | **～150** | 🎯 |
| 7日連続 | **500** | 🔥 |
| 30日連続 | **2000** | 💪 |

### レベル一覧

```
🌱 Level 1: Newbie
   0 - 99 ポイント
   初心者レベル

🔰 Level 2: Beginner  
   100 - 499 ポイント
   基礎を学んだ

⚡ Level 3: Intermediate
   500 - 999 ポイント
   中級レベル

🌟 Level 4: Advanced
   1,000 - 4,999 ポイント
   上級者

💎 Level 5: Expert
   5,000 - 9,999 ポイント
   エキスパート

👑 Level 6: Master
   10,000+ ポイント
   マスターレベル！
```

### ポイント計算式

```javascript
総ポイント = (解決Issue数 × 10) 
           + (マージPR数 × 20) 
           + (コミット数 × 1)
           + ボーナスポイント
```

## 🏆 実績システム

### Issue関連

- 🎯 **First Blood** - 最初のIssue解決
- ⭐ **Issue Hunter** - 10個のIssue解決
- 🏅 **Issue Master** - 50個のIssue解決
- 👑 **Issue Legend** - 100個のIssue解決

### PR関連

- 🚀 **First Flight** - 最初のPRマージ
- 💎 **PR Pro** - 10個のPRマージ
- 🌟 **Merge Wizard** - 50個のPRマージ

### コミット関連

- 🔥 **Commit Streak** - 100コミット
- 💪 **Code Warrior** - 500コミット

### 特別実績

- ✨ **Perfectionist** - 80%完了率達成
- 🤝 **Team Builder** - 5人以上の貢献者

### マイルストーン実績

特定のマイルストーンに達すると、特別なお祝いIssueが自動生成されます！

- 10個目のIssue解決
- 50個目のIssue解決
- 10個目のPRマージ

## 🎯 デイリークエスト

### 仕組み

- **発生**: 毎朝9:00 JST
- **有効期限**: 24時間
- **報酬**: 30～150ボーナスポイント

### クエスト種類 (10種類)

1. 🐛 **Bug Hunter**
   - バグを修正
   - 報酬: 50ポイント

2. ✨ **Feature Friday**
   - 新機能追加
   - 報酬: 100ポイント

3. 📝 **Documentation Day**
   - ドキュメント改善
   - 報酬: 30ポイント

4. 🔒 **Security Sentinel**
   - セキュリティ強化
   - 報酬: 150ポイント

5. ♻️ **Refactor Master**
   - リファクタリング
   - 報酬: 75ポイント

6. ⚡ **Performance Boost**
   - パフォーマンス最適化
   - 報酬: 120ポイント

7. 🤖 **Automation Expert**
   - 自動化追加
   - 報酬: 90ポイント

8. 🎯 **Issue Closer**
   - 3つのIssue解決
   - 報酬: 80ポイント

9. 🚀 **Deploy Day**
   - mainブランチにマージ
   - 報酬: 60ポイント

10. 🤝 **Team Player**
    - 他の人を支援
    - 報酬: 70ポイント

### ストリークボーナス

```
🔥 7日連続:  +500ボーナスポイント
💪 30日連続: +2000ボーナスポイント + 特別バッジ
⭐ 週間全制覇: +300ボーナスポイント
```

### クエスト確認方法

1. [DAILY_QUEST.md](DAILY_QUEST.md) を確認
2. `daily-quest` ラベルのIssueを確認
3. Actionsタブで実行履歴を確認

## 📊 統計ダッシュボード

### [GAME_STATS.md](GAME_STATS.md)

ここで以下が確認できます：

- 🏆 **現在のレベル**
- 📊 **総ポイント**
- 📈 **統計**:
  - Issue総数/解決数/オープン数
  - PR総数/マージ数
  - コミット総数
  - 貢献者数
- 🎯 **完了率** - プログレスバー付き
- ✨ **アンロック済み実績**
- 🎯 **次のマイルストーン**

### 更新頻度

統計は以下のタイミングで自動更新：

- Issue作成/解決時
- PR作成/マージ時
- mainブランチへのプッシュ時

## 🎮 使い方

### 1. ポイントを稼ぐ

```
1. Issueを作成/解決 (+10ポイント)
2. PRを作成/マージ (+20ポイント)
3. コミットを作成 (+1ポイント)
4. デイリークエスト完了 (+ボーナス)
```

### 2. 実績をアンロック

```
1. 統計が自動更新
2. 実績条件達成
3. GAME_STATS.mdに実績追加
4. マイルストーンでお祝いIssue生成
```

### 3. レベルアップ

```
1. ポイント稼ぐ
2. レベル閾値達成
3. 自動でレベルアップ
4. GAME_STATS.mdで確認
```

### 4. デイリークエスト

```
1. 毎朝9時にクエスト発生
2. DAILY_QUEST.mdで確認
3. クエスト完了
4. Issueをクローズで報酬獲得
```
